Alembic Android App

TOOLS: Android, Java, MySQL, SQLite, PHP, JSoup

Demo available here

Scrapes product info off an alt-goth perfume site and lets users rate scents they've tried. Features a recommendation agent that recommends new scents based on the ingredients of the scents they've rated highly.

Created for a Mobile Development class (but mostly because I want to be able to use that recommendation agent).

TwitListed Twitter App

TOOLS: Python, Tweepy, Flask, Git

Code available here

Calculates the average tweet frequency of every Twitter account you follow then creates and sorts them into four new Twitter lists:

  • Accounts who tweet 30 or more times per day
  • Accounts who tweet between 5 and 30 times per day
  • Accounts who tweet between 1 and 5 times per day
  • Accounts who tweet less than once per day

Access Map Seattle Web Application

TOOLS: HTML, CSS, Javascript, Leaflet, Github

Publicity is here

Creepy Tetris

TOOLS: Java Swing, Subversion, Eclipse

Demo available here

In addition to the normal Tetris mode, includes an optional Creepy Tetris mode with a story introduction and conclusions that vary based on win or loss state, dynamic wallpaper that changes as the game progresses, atmostpheric music and ghostly sound effects, and custom voice recordings indicating increasing distress as the blocks grow higher and relief when lines are cleared.

Created as a class assignment where the extra credit option was to "add one additional feature to the basic Tetris game".

Netsuite Automation Script

TOOLS: Python, Selenium

Automates operation of a customer relationship management website based on input from an Excel file, thereby saving significant time and effort in resolving a large number of identical cases.

Created to solve a problem where a computer glitch had caused thousands of nonsense customer emails to enter a call center's customer relationship management site, creating thousands of unresolved cases which interfered with accurate reporting. Each case had to be individually opened and required fields appropriately selected before the case could be closed.

RPG Character Creator

TOOLS: Visual Basic, Visual Studio

Code available here

Guides creation of a character for an unpublished tabletop gaming system. Rules, equipment, and skills are imported from .csv files. Modifies stats as skills are selected for training or specialization and equipment is purchased and worn.

Created following my first programming class, for fun and to replace the 12 page, multi-tab spreadsheets otherwise required for managing characters in a friend's homemade tabletop role playing game.

Notification Android App

TOOLS: Android, Java, Git

At a user-selected frequency, displays the odds of a random unlikely event paired with the odds of being in a distracted driving crash. Displays notifications only while phone is in use and not while locked or in a rest state, to avoid causing distraction while driving.

Created for and won 2nd place at the 2014 UW Seattle chapter of the International Women's Day Hackathon.
